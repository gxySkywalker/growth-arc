# 背景音乐与音效指南

成长轨迹的音频系统采用用户自定义方案——软件本身不含任何版权音频，由你放入自己喜欢的音乐。

## 目录结构

```
assets/audio/
├── bgm/                 ← 背景音乐（循环播放）
│   ├── cottage.mp3      ← 炉火小屋
│   ├── expedition.mp3   ← 远征途中
│   └── .gitkeep
└── sfx/                 ← 音效（单次播放，v0.2 接入）
    └── .gitkeep
```

## 文件要求

| 项目 | 要求 |
|------|------|
| 格式 | MP3 / WAV / OGG |
| 命名 | 必须与上表完全相同 |
| BGM | 建议 1-3 分钟循环段 |
| 音效 | 建议 1 秒以内短促音 |

## 如何获取音乐

- 从你合法拥有的游戏中提取
- OpenGameArt.org（筛选 CC0 许可）
- Freesound.org（筛选 CC0 许可）
- itch.io 游戏资产包（免费/付费）
- Suno AI / Udio 生成（注意商用条款）

## 试听推荐

以下关键词可帮你搜到类似风格的 CC0 音乐：

- `GBA chiptune town theme`
- `pixel RPG cottage music`
- `8-bit adventure calm loop`
- `chiptune fireplace ambient`

---

**注意：此目录的音频文件不会被上传到 GitHub。** 请勿将任何你不拥有版权或未获授权的音乐文件提交到仓库。
