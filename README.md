# 成长轨迹 (Growth Arc)

一个只属于你、数据保存在本地的像素 RPG 生活记录软件。在温暖的中世纪边境世界中小小地生活，每一次真实专注都会让世界向前生长一步。

## 已实现

### ✉️ 天使邮局

邮局二楼住着一位小天使。每天傍晚，她会在灯下整理你的旅途，写下一封温暖的信。

- **每日星笺** — 每次远征后自动生成，空白日不生成，离线自动补发
- **旅途札记** — 每完整一周结束后回顾旅途
- **归灯节来信** — 每年 11 月 7-16 日，边境小镇迎来归灯节
- **生日纪念信** — 设置生日后每年自动收到
- **欢迎信** — 第一次进入邮局时，小天使会亲手写一封欢迎信
- **AI 信件（可选）** — 填写 DeepSeek / OpenAI Key 后，小天使可以亲手写更特别的信；连通测试、每日星笺与每周札记共用同一条安全链路。不填也完全不影响使用

### 🐾 伙伴羁绊系统

- 伙伴营地升级为同行图鉴与归途档案
- 羁绊章节：初识 → 同行 → 长成
- 羁绊达到阈值后自动触发成长事件
- 伙伴个体差异：14 种性格 + 物种习惯 + 小毛病
- 栗子拥有独立的旅伴档案和共同记忆

**八位同行伙伴**：
- 栗子：炉尾 → 栗鬃 → 炭尾 / 松影 / 月爪
- 枝绒 → 苔亚 → 森冠
- 灯团 → 星烛 → 夜璃
- 涟牙 → 漪爪 → 湾澜
- 小石獾 → 岩甲獾 → 铠獾王
- 暮羽子 → 咕夜枭 → 冥翔鹰鸮
- 小丘 → 云丘兔 → 风茸旅兔
- 小火牙 → 赤翼龙 → 余烬古龙

每位伙伴都有独立的三阶段视觉、共同记忆与小屋呈现；成长表达同行关系的变化，而不是数值强化。

### 🏠 炉火小屋

- WASD / 方向键走动的像素主角
- 壁炉火光、物件碰撞、可交互家具和常伴伙伴
- 一键开始远征

### 🧭 远征与结算

- 自由正计时，支持暂停、休眠恢复和异常保护
- 多路标结算，沿途经验递减，单场上限 30 XP
- 远征类型：brief / short / expedition / deep
- 四级掉落：普通、罕见、稀有、珍稀，共 11 件旅途物品
- 定向伙伴纪念物、夜间罗盘、单次星辉加成与旅者银铃相遇状态
- 尚未开放的交易、地图、药汤、火炉合成与 NPC 送礼物品会安全保留在收藏中

### 🎮 键盘导航

- 侧边栏、天使邮局、天文台支持完整键盘操作
- RPG 风格选择光标（▶ 像素闪烁指示器）
- 炉火小屋与菜单系统互不干扰

### 🌍 世界状态

- 邮局窗框会响 → 艾达来修
- 窗台的花会发芽、开花、凋谢
- 归灯节期间镇上挂灯
- 小天使的生活细节自然出现在信件中

### 📊 天文台

- 日/周双视图，24 小时星图
- 7×24 星轨热力图、七日柱图
- 观测札记
- 修复异步加载时图表容器未初始化导致的偶发空白，并避免旧请求覆盖当前日期或标签页

### 🗺️ 制图室

- 三级结构：旅途方向 → 路线阶段 → 路标
- 拖拽排序、12 色拾色器

### 🔒 隐私与数据

- SQLite 本地存储，数据只在你自己的电脑上
- API Key 使用 Windows DPAPI 加密存储
- 系统托盘驻留，90 分钟温和提醒
- 不上传、不同步、不自动备份

## 快速开始

**安装包**：从 [GitHub Releases](https://github.com/gxySkywalker/growth-arc/releases) 下载最新版本安装。

**开发模式**：

```powershell
npm run dev
```

**测试与构建**：

```powershell
npm test        # 231 tests
npm run build
npm run dist    # Windows NSIS 安装包
```

---

AI 信件为可选能力。在旅人手册中填写信笺钥匙即可启用；没有钥匙时小天使会用准备好的信笺，一样温暖。
