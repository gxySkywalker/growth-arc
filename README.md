# 成长轨迹

一个只属于你、数据保存在本地的游戏化学习系统。在温暖的中世纪边境世界中小小地生活，每一次真实专注都会让世界向前生长一步。

## 当前版本

**v0.2.0 — 远征返程与多路标结算**（2026-07-19）

## 已实现

### 🏠 炉火小屋
- WASD / 方向键走动的像素主角，4 方向朝向
- 壁炉火光、物件碰撞、可交互家具和常伴伙伴
- 一键开始远征，专注结束后自动回到小屋

### 🧭 远征与结算
- 自由正计时，支持暂停、休眠恢复和异常保护
- 一次远征可同时抵达主要路标与多个沿途路标
- 沿途经验按顺序递减结算，单场上限 30 XP
- 有效专注不足 5 分钟时保留记录，不追加经验
- 远征途中自动播放背景音乐

### 📖 旅途编年史
- 按日期分组的时间线，每次远征自动归档
- 展开式完整档案：掉落、沿途抵达、知识遗物、新伙伴
- 不足一分钟的专注按秒显示

### 🗺️ 制图室
- 三级结构：旅途方向 → 路线阶段 → 路标
- 拖拽排序、12 色拾色器、右键菜单
- 路标支持完成、归档、删除和重新打开

### 📊 成长系统
- 经验等级、一次性完成奖励、九项成就
- 日复盘、周统计、AI 结构化学习报告（OpenAI / DeepSeek）

### 🎒 物品与伙伴
- 9 种远征掉落，支持使用效果和背包管理
- 8 种同行伙伴，羁绊积累、阶段进化和进化分支选择

### 🔒 隐私与数据
- SQLite 本地存储，数据只在你自己的电脑上
- Windows DPAPI 加密保护 API Key
- 系统托盘驻留，90 分钟温和提醒

## 快速开始

**安装包**：从 [GitHub Releases](https://github.com/gxySkywalker/growth-arc/releases) 下载最新版本安装。

**开发模式**：

```powershell
npm run dev
```

**测试与构建**：

```powershell
npm test
npm run build
npm run dist
```

AI 报告为可选能力。请在设置页填写 API Key；没有 Key 时所有功能仍可完整使用。

数据目录可从设置页直接打开。当前版本不会上传、同步或自动备份数据库。
